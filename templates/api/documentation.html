Usage for Grabbing Attendance


Starter code to get attendance information 

  
  import requests

  #credentials
  username = student_username
  password = student_password

  #grab information for attendance
  params = {"username":username,"password":password, "token":token}
  params = json.dumps(params)
  headers = {'Content-Type': 'application/json'}
  attedance_request = requests.post("https://api.studysquawk.tech/attendance",  data=params, headers=headers)

  #converts output to a json format(dictionary)
  attendance_data = attedance_request.json()
  #grabs data from dictionary
  attendance = attendance_data['attendance']

Usage for Grabbing Both


Starter code to getting both the attendance and the grade information 

  
  import requests

  #credentials
  username = student_username
  password = student_password

  #grab information for attendance
  params = {"username":username,"password":password, "token":token}
  params = json.dumps(params)
  headers = {'Content-Type': 'application/json'}
  attedance_request = requests.post("https://api.studysquawk.tech/both",  data=params, headers=headers)

  #converts output to a json format(dictionary)
  attendance_data = attedance_request.json()
  #grabs data from dictionary
  attendance = attendance_data['attendance']

Using the Code above, you will get the same keys and responses of both Attendance and the Grades. Access them with the same keys and the same way. The Both Function, just return both of the information in one package.
